<h2>Unity での設定</h2>
<article>
  インスペクターから、ゲームシーン内 "GameManager" にアタッチされている 、"SerialHandler.cs" の "PortName" を、使用するPCのM5本体が接続されているポートに設定してください。<br>
  Arduinoで既にPCとM5が接続したことがある場合、その時使用したポートを設定します。
</article>
